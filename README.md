### Run Locally 


**Clone the repo:**

```bash
git clone
```

**Install the dependencies:**

```bash
npm install
```

**Setup .env file**

```bash
VITE_APP_ID = your_edamam_app_id;
VITE_APP_KEY = your_edamam_app_key;
```

**Start the app**

```bash
npm run dev
```
